## Generator Changes (`generator/generate.py`)

Four new capabilities added:

1. **`bin_mkpath`** — when config has `"bin_mkpath": true`, emits `    bin.mkpath` + blank line in install block before build_steps.
2. **Bin test assertions** — `{"type": "bin", "path": "X"}` in test_paths → `    assert_path_exists bin/"X"`.
3. **`header_comment` + `deprecate`** — `header_comment` emits `# <line>` per entry immediately before the class line (preserving leading whitespace for indented code blocks); `deprecate` emits `  deprecate! date: "<date>", because: "<reason>"` after `license "MIT"` with a blank line before.
4. **`{command_count}` token** — replaced in desc, install_script_summary, install_script_desc, caveats_extra at generation time via `apply_command_count_token()`.

Additional mechanism: **`libexec_copy_map_after`** — interleaves `cp_r` calls immediately after a specific `libexec_copy_map` entry (used by himalaya-mcp to place hooks between plugin.json and marketplace.json copies, matching the committed file's order).

# Data-Wise Homebrew Tap

Homebrew formulae for Data-Wise tools.

## Install

```bash
brew tap data-wise/tap
brew install examify
```

## Packages

| Formula | Description |
|---------|-------------|
| **examify** | Create exams from Markdown and export to Canvas QTI format |
